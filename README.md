# Nord JS

A [Nord theme](https://www.nordtheme.com/) port for JavaScript/TypeScript

## Installation

```bash
npm i --save nord-js
```

## Usage

### CommonJS/Node

```javascript
const Color = require("nord-js");
```

### ESM/TS

```javascript
import * as Color from "nord-js";
```

### Browser

```html
<script src="/path/to/index.js"></script>
```
